Fluxo Completo — Do Cadastro à Postagem

Campos do Cadastro de Imóvel

Obrigatórios
  • Fotos (até 20, arrastar e soltar)
  • Tipo (apartamento, casa, terreno, comercial)
  • Operação (venda, aluguel, temporada)
  • Preço (€)
  • Cidade + Bairro
  • Área (m²)
  • Quartos / Banheiros
Opcionais (enriquecem o post)
  • Descrição livre (IA complementa)
  • Andar / Elevador
  • Garagem / Piscina / Varanda
  • Ano de construção
  • Certificado energético
  • Vídeo do imóvel
  • Redes para publicar (default: todas)
